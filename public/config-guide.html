🤖 AI 工具配置指南

📍 您的代理端点

OpenAI 兼容: https://your-proxy.vercel.app/v1

Anthropic 兼容: https://your-proxy.vercel.app/anthropic

Claude Code 配置

VS Code Claude Code 扩展

在 ~/.claude/settings.json 中添加以下配置：

{
  "env": {
    "ANTHROPIC_BASE_URL": "https://your-proxy.vercel.app/anthropic",
    "ANTHROPIC_AUTH_TOKEN": "your-minimax-api-key",
    "API_TIMEOUT_MS": "3000000",
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": 1,
    "ANTHROPIC_MODEL": "MiniMax-M2.5",
    "ANTHROPIC_SMALL_FAST_MODEL": "MiniMax-M2.5",
    "ANTHROPIC_DEFAULT_SONNET_MODEL": "MiniMax-M2.5",
    "ANTHROPIC_DEFAULT_OPUS_MODEL": "MiniMax-M2.5",
    "ANTHROPIC_DEFAULT_HAIKU_MODEL": "MiniMax-M2.5"
  }
}

Cursor AI 配置

Cursor 设置

在 Cursor 中配置 MiniMax API：

  1. 打开 Settings → Models
  2. Override OpenAI Base URL: https://your-proxy.vercel.app/v1
  3. Add Custom Model: MiniMax-M2.5
  4. 输入您的 API Key

TRAE 配置

TRAE IDE

在 TRAE 中添加 MiniMax 模型：

  1. 点击 Settings 图标 → Models
  2. 点击 + Add Model
  3. Provider: MiniMax-Global
  4. Model: MiniMax-M2.5
  5. API Key: 您的 MiniMax API Key

OpenCode 配置

OpenCode CLI

编辑 ~/.config/opencode/opencode.json：

{
  "$schema": "https://opencode.ai/config.json",
  "provider": {
    "minimax": {
      "npm": "@ai-sdk/anthropic",
      "options": {
        "baseURL": "https://your-proxy.vercel.app/anthropic/v1",
        "apiKey": "your-minimax-api-key"
      },
      "models": {
        "MiniMax-M2.5": {
          "name": "MiniMax-M2.5"
        }
      }
    }
  }
}

Kilo Code / Cline / Roo Code 配置

VS Code AI Coding 扩展

在这些扩展中设置：

  • API Provider: MiniMax
  • MiniMax Entrypoint: your-proxy.vercel.app
  • MiniMax API Key: 您的 API Key
  • Model: MiniMax-M2.1

Zed 配置

Zed Editor

在 Zed 的 Agent 面板中：

  1. 打开 Agent Settings
  2. LLM Provider → +Add Provider → OpenAI
  3. API URL: https://your-proxy.vercel.app/v1
  4. API Key: 您的 API Key
  5. Model Name: MiniMax-M2.5

⚠️ 重要提示

  • 将 your-proxy.vercel.app 替换为您的实际 Vercel 部署域名
  • 将 your-minimax-api-key 替换为您的实际 MiniMax API Key
  • 建议在本地配置文件（如 ~/.claude/settings.json）中设置 API Key，而不是提交到代码仓库

💡 获取 MiniMax API Key

访问 MiniMax Developer Platform 获取您的 API Key。如果您有 Coding Plan 订阅，请在 Token Plan 页面获取专用 API Key。
